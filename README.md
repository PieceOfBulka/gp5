# Задача 1: Предсказание популярности трека

## Данная задача часть общего пайплана и является одним из первых этапов

**Работаем с табличными данными для предсказания популярности от 0 до 100.** 

- 0 - трек точно не "взлетит"
- 100 - трек будет очень популярен.


**Датасет для работы:** https://www.kaggle.com/datasets/saichaitanyareddyai/spotify-tracks-dataset-audio-features?resource=download


**Датасет заносится в общий git-проект, поскольку его размер не является критичным для работы (размер файла = 19,4 МБ < 50 МБ)**

## Структура проекта

```
.
|– src
    |— BERTnCNN — ноут с пайплайном обучения берта и TextCNN
    |— utils — папка с полезными инструментами (в т.ч. загрузка датасетов в ClearML)
    |— eda — папка с EDA
    |— data — папка с данными
|— requriments.txt — папка с необходимыми зависимостями
```
